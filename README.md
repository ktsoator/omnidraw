# omnidraw

A fast, thread-safe draw engine in Go for events, giveaways, and gamified campaigns.

## Demos

Explore functional examples of Omnidraw in action:

- **[Master Demo](./demos/master/README.md)**: A full-stack lucky draw implementation featuring a RESTful API server (Gin) and a premium interactive CLI client. Features O(1) draw performance and automatic player deduplication.
